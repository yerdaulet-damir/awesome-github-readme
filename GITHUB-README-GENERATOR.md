## How the Generator Works

**[coolreadme.xyz/generator](https://coolreadme.xyz/generator)** — type your username, pick a card type, adjust options, copy the markdown. One URL is the entire card.

```markdown
<!-- This is all you paste into your README -->
![Card](https://coolreadme.xyz/api/cinematic?user=YOURUSERNAME&status=SHIPPING+CODE&focus=What+you+build)
```

GitHub fetches the URL when rendering your profile. The card generates live. No webhook, no token, no CI step.

# Templates

Ready-to-use README templates. Copy, customize, done.

## Available Templates

| Template | Style | Tools Used |
|----------|-------|------------|
| [01-minimal.md](01-minimal.md) | Clean, professional | stats, badges |
| [02-full-featured.md](02-full-featured.md) | Complete profile | typing, stats, trophies, snake |
| [03-oss-maintainer.md](03-oss-maintainer.md) | Project-focused | stats, sponsor badges |
| [04-creative-personal.md](04-creative-personal.md) | Personality | spotify, gaming, stats |
| [05-coolreadme-cards.md](05-coolreadme-cards.md) | Cinematic cards | coolreadme.xyz (Netflix, AI, YouTube, etc.) |

## How to Use

1. Choose a template
2. Replace `YOURNAME` with your GitHub username
3. Customize text and colors
4. Copy to your profile README

## Customization

### Colors

Common accent colors:
- `A78BFA` - Purple
- `00FF88` - Green
- `3B82F6` - Blue
- `EF4444` - Red

### Themes

Most tools support themes:
- `dark` - Dark mode
- `radical` - Purple dark
- `darkhub` - GitHub dark

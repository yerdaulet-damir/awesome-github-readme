## How It Works

Your pet **evolves automatically** based on your GitHub commit streak. Start coding, keep your streak alive, and watch your pet grow from a baby to a legend.

Just paste one line of markdown into your README:

```markdown
![My Pet](https://coolreadme.xyz/api/fox-card?user=YOURUSERNAME)
```

That's it. The card reads your real GitHub streak and renders the matching stage.

### Evolution Stages

Your streak determines your pet's stage. Each animal has unique breeds and appearances per stage.

| Stage | Streak Days | Effects | Vibe |
|-------|:-----------:|---------|------|
| **Seed** | 0–3 | None | Just hatched |
| **Sprout** | 4–9 | None | Getting consistent |
| **Grind** | 10–19 | Bounce | Locked in |
| **Streak** | 20–29 | Bounce + Sparkles | Unstoppable |
| **God** | 30–49 | Crown + Fire | Ascended |
| **Legend** | 50–99 | Halo | Legendary status |
| **Mythic** | 100–199 | Rainbow | Beyond mortal |
| **Eternal** | 200–364 | Lightning | Time itself bows |
| **Transcended** | 365+ | Galaxy background | You ARE the code |

## Customization

All pet cards support these query parameters:

| Parameter | Default | Description |
|-----------|---------|-------------|
| `user` | `yourname` | Your GitHub username (auto-fetches streak) |
| `streak` | *auto* | Override current streak days |
| `longest` | *auto* | Override longest streak |
| `total` | *auto* | Override total contributions |
| `accent` | *per stage* | Custom glow color (hex, e.g. `%2300FF88`) |
| `bg` | `dark` | Background: `dark`, `midnight`, or `deep` |
| `style` | `default` | Layout: `default` (800×400), `compact` (800×280), `minimal` (800×200) |
| `breed` | *per stage* | Force a specific breed (e.g. `god`, `nine-tail`, `emperor`) |
| `hideTotal` | `false` | Hide total contributions count |

### Examples

**Compact style with custom accent:**
```markdown
![Pet](https://coolreadme.xyz/api/fox-card?user=YOURUSERNAME&style=compact&accent=%23FF6600)
```

**Force a specific breed:**
```markdown
![Pet](https://coolreadme.xyz/api/dog-card?user=YOURUSERNAME&breed=god)
```

**Midnight background:**
```markdown
![Pet](https://coolreadme.xyz/api/penguin-card?user=YOURUSERNAME&bg=midnight)
```

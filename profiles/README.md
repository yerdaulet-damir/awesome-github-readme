## How Profiles Are Displayed

Each profile file shows:

1. **Live Preview** - The actual rendered README (what people see on GitHub)
2. **Tools Used** - Which tools from our collection
3. **Links** - To the real GitHub profile
4. **Source Code** - Collapsible section with the markdown
